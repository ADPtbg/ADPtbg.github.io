Bienvenido al Administrador de Parcelas

El Administrador de Parcelas, desarrollado por Tomás Bertin desde 2022, es una herramienta informática (Software) que ayuda a los trabajadores rurales a organizar y gestionar la información de sus parcelas, ganado y pastura de manera efectiva.

Lo nombrado anteriormente se logra a través de un software programado en Microsoft Visual Basic en lenguaje de programación Java .NET, también con una buena bases de datos MySql y phpMyAdmin, que esta es una herramienta que tiene como intención manejar la administración de MySql a través de páginas web, utilizando un navegador (Google Chrome, Mozilla, etc...), y para iniciar todos estos servicios se utilizó WAMPSERVER el cual consta con: Servidor Web Apache, OpenSSL, MySql y lenguaje de programación PHP

Características del software

  • Gestión de información de parcelas
  • Registro de datos de ganado
  • Seguimiento de la calidad de pastura
  • Almacenamiento de datos en una base de datos MySQL

Este software proporciona una solución integral para mejorar la organización en el campo y facilitar la toma de decisiones informadas.
